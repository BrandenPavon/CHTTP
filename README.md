CHTTP

Getting Started (Debian-based distros)
---------------
1. Install requirements
    ```sh
    sudo apt update
    ```
    ```sh
    sudo apt install build-essential
    ```
2. Clone the repo:
    ```sh
    git clone https://github.com/BrandenPavon/CHTTP.git
    ```

Description:
    HTTP Server in C

Build:
  make
Build clean:
  make clean
Run:
  ./main
