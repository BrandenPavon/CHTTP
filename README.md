
# CHTTP

## Description
HTTP server written in C.

## Requirements (Debian-based systems)

```sh
sudo apt update
sudo apt install build-essential

## Installation

```sh
git clone https://github.com/BrandenPavon/CHTTP.git
cd CHTTP
```

## Build

```sh
make
```

## Clean

```sh
make clean
```

## Run

```sh
./main
```

## Server

Runs on:

```
127.0.0.1:8089
```
